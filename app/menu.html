Tipos de Gráficos

Gráfico Radar

Un gráfico radar permite comparar varias variables de un jugador o equipo en forma de "araña". Cada eje representa una estadística y la forma resultante muestra fortalezas y debilidades de manera visual.

Gráfico de Caja

El gráfico de caja (boxplot) muestra la distribución de una estadística: valores mínimos, máximos, mediana y cuartiles. Es ideal para comparar el rendimiento de un jugador respecto al resto.

Gráfico de Dispersión

El gráfico de dispersión muestra la relación entre dos variables (por ejemplo, goles vs. asistencias) para todos los jugadores, permitiendo identificar tendencias y outliers.
